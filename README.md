# Generative AI Research Project

This repository contains organised research data for the project "Using Generative AI Tools - Boon or Bane".

## Folder Structure

- Data/
  - Literature: research papers and notes
  - Quantitative: survey data and analysis scripts
  - Qualitative: interview transcripts and consent forms
  - Media: images and other media files

- Reports/
  - Drafts: draft reports
  - Final: final reports

## Purpose

The repository is structured to support efficient data management, version control, and collaboration.
